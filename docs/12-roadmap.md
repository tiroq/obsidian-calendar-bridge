# 12 — Roadmap (MVP → v1.0 → vNext)

## MVP (0.1)
- Source: ICS (secret/public)
- Horizon sync: create notes for next 3 days
- Template rendering + AUTOGEN blocks
- Basic subscriptions by seriesKey (based on iCal UID)
- Manual sync command

## 0.2
- Series select modal
- Series pages generation (index)
- Prev/Next links (within window)
- Dry-run preview

## 0.3
- Google API source (OAuth) desktop first
- calendar picker
- richer fields (attendees, conference)

## v1.0
- stable state migrations
- polished UX, docs, publishing checklist
- robust edge-case handling (cancel/move)

## vNext
- state-in-vault (sync between devices)
- rules-based virtual series
- redaction mode
- optional LLM agenda suggestions (explicit opt-in)
